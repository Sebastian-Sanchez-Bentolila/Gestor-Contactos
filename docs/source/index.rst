Bienvenido a la documentacion del Gestor de Contactos!
======================================================
**Gestor de contactos** es una apliación de escritorio que permite agendar tus contactos
en una base de datos personal. Es una herramienta *facil e intuitiva* de usar. Esta contruido como *modelo-vista-controlador*.


Visite el proyecto en `GitHub <https://github.com/Sebastian-Sanchez-Bentolila/Gestor-Contactos>`_ y sígueme en mis redes sociales.


Intagram: `@sebas_sanchez_bentolila <https://www.instagram.com/sebas_sanchez_bentolila>`_.

.. note::
   Este proyecto está en desarrollo activo.

Características
---------------
Esta construido con *POO* en Python y tiene con una base de datos local Sqlite3.
Permite hacer lo siguiente:


1. **Crear, Eliminar y Modificar Contactos:**

   + Permite agregar nuevos contactos a la base de datos.

   + Proporciona funcionalidades para eliminar contactos existentes.

   + Permite la modificación de la información de los contactos.
|
2. **Buscar Usuarios por Nombre:**

   + Ofrece una función de búsqueda que permite encontrar usuarios por su nombre.
|
3. **Ver los Contactos Agendados de Forma Organizada:**

   + Proporciona una interfaz organizada para visualizar la lista de contactos almacenados en la base de datos.
