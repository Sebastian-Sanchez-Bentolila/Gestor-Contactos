Introducción
============

El proyecto es un gestor de contactos desarrollado en Python. Proporciona una interfaz gráfica de usuario (GUI) para que los usuarios gestionen su lista de contactos, con características como agregar, buscar, modificar y eliminar contactos.
Esta estructurado con un patron MVC, y un patron Observador (quien se encarga del archivo log.txt). Posee ademas una funcionalidad de cliente-servidor.
